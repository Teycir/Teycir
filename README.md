## About

I build static analysis tooling and exploit proof-of-concepts for **ZK circuits** and **EVM smart contracts**, with a focus on pattern-based vulnerability detection across the full ZK stack — from Circom/Noir constraints to Solidity logic.

When I'm not researching ZK security, I work on web/API penetration tooling and privacy-first cryptographic applications.

## Research & Tooling

### 🔐 ZK Security Research
Pattern-based detection across ZK circuit layers (Circom · Noir · Cairo · Halo2) and the EVM.

| Project | Description |
|---|---|
| [ZkPatternFuzz](https://github.com/Teycir/ZkPatternFuzz) | Multi-backend ZK circuit fuzzer — YAML-driven pattern detection engine in Rust |
| [ZkPatternMatcher](https://github.com/Teycir/ZkPatternMatcher) | Structural pattern matching for ZK constraint systems |
| [smartcontractpatternfinder](https://github.com/Teycir/smartcontractpatternfinder) | YAML-pattern static analysis for Solidity/EVM — same methodology, contract layer |
